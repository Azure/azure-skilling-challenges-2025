# Azure Ignite Challenges

Welcome to the Azure Ignite Challenges landing page, featuring 9 hands-on learning tracks.

This landing page aggregates 9 Ignite-aligned learning challenges:

1. **Innovate with AI Agents** - Azure AI Agent Service, multi-agent orchestration, agent frameworks  
2. **Build AI Apps with Azure** - Azure OpenAI Service, RAG patterns, prompt engineering  
3. **Fabric Data Engineering** - Microsoft Fabric pipelines, OneLake, ETL/ELT patterns  
4. **Fabric Data Analytics** - Real-time analytics, Power BI integration, KQL and SQL analytics  
5. **Data Fundamentals** - Relational and non-relational databases, data warehousing basics  
6. **AI Security & Compliance** - Microsoft Defender, identity management, Zero Trust architecture  
7. **Agentic DevOps with GitHub Copilot** - GitHub Copilot, Agent Mode, Spaces, prompt engineering  
8. **Migrate to Azure** - Cloud migration strategies, Azure Migrate tools, database migration  
9. **Modernize on Azure** - AKS, microservices, cloud-native patterns, DevSecOps
